GitHub Pages for dette prosjektet
=================================

1. Opprett et nytt repository på GitHub (uten README hvis du pusher eksisterende mappe første gang).

2. Fra prosjektmappen (PowerShell):

   git remote add origin https://github.com/DITT-BRUKERNAVN/DITT-REPO.git
   git branch -M main
   git push -u origin main

3. På GitHub: Repo → Settings → Pages → Build and deployment
   - Source: "Deploy from a branch"
   - Branch: main, folder: / (root)

4. Etter noen minutter: https://DITT-BRUKERNAVN.github.io/DITT-REPO/

Merk: Nettsiden bruker relative stier (css/, js/, data/) — det fungerer under /repo-navn/ på Pages.

Instagram-sync: legg secrets INSTAGRAM_ACCESS_TOKEN og INSTAGRAM_USER_ID under
Repo → Settings → Secrets and variables → Actions (ikke Pages).
